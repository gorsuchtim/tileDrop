TileDrop 2.0

Score: 0

Streak: 0

Object of the Game

  • Score as many points as possible before all of the blocks disappear from the gameboard

Dropping Blocks

  • At the start of the game 1 block disappears with each beat
  • At a score of >= 1000 and every 100pts additional blocks drop per beat

Scoring points and replacing blocks

  • Tapping a white flash scores 100 points and replaces 1 block to the board

Awesomemeter!

  • The Awesomemeter fills by 1/10th for each white flash tap scored
  • When the Awesomemeter is filled multiple blocks are replaced

SyncFlash!

  • At random intervals the flash will be green. A green SyncFlash scores 200pts
  • The SyncFlash sequence lasts a randomly selected number of flashes. Tapping all SyncFlashes brings all blocks back to the game board
