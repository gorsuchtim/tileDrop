Object of the Game

  • The game begins with x number of blocks (x rows and y columns)
  • When the game starts blocks will be removed with the beat of the music
  • At the start of the game only 1 block is removed with each beat
  • The player's goal is to score as many points as possible before all blocks are removed

Scoring points and replacing blocks

  • A white flash appears with each beat of the music
  • If the player taps the flash before it disappears, 1 block is replaced on the game board
    • There is no consequence for failing to tap a block, except that another block has fallen without being replaced
  • With each tap the player scores 100 points

Powerups

Awesomemeter!

  • With each white flash (not SyncFlash) tap, the Awesomemeter at the top of the screen will fill by 1/10th (10%)
  • When the Awesomemeter is filled, blocks are replaced. The number of blocks replaced is equal to 1/10th (10%) of the player's score

SyncFlash!

  • At random intervals, between 7 and 15 white flashes, the flash will turn green
  • Tapping a SyncFlash flash will score the player 3 points per tap
  • The green SyncFlashes last between a randomly selected interval of 5 and 10 flashes
  • If the player successfully taps all green flashes (not missing any) then all of the blocks re-appear on the game board
